# **App Name**: Verse Exam Prep

## Core Features:

- Question Display: Display question text, options, and explanation.
- Test Creation: Create new tests with details, including duration, pricing, and associated hierarchy.
- User Authentication: Enable user login, registration, and role assignment (admin/student).
- Test Access Control: Control student access to tests based on enrollment or purchases.
- Test Results: Calculate test results with marks and keep record of the attempt with its final grade.
- Content Generation: An AI tool helps automatically generate a relevant test question.  It selects the topic of the test question automatically based on an understanding of the test requirements.
- Progress Tracking: Keep record of a student's performance by exam type or content area.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), reflecting trust and knowledge.
- Background color: Light blue (#E8EAF6), to create a calm, learning environment.
- Accent color: Purple (#7E57C2), highlighting interactive elements and calls to action.
- Body and headline font: 'Inter', sans-serif, for clean and modern readability.
- Use modern and consistent icons for navigation and test-taking actions.
- Clean, structured layouts optimized for both phones and tablets.
- Subtle animations during test transitions to keep the test-taker focused and calm.
